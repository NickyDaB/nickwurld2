IGME.235 - Intro to Game Web Tech

Rochester Institute of Technology · Undergraduate · Core Class

An introduction to web technologies for interactive media and games. Topics include HTML, CSS, JavaScript, browser rendering, input handling, and simple browser-based game development.

  • Browser game case studies (Pong, Asteroids-like projects)
  • Emphasis on iteration, debugging, and user testing
  • Final project: small original web game
